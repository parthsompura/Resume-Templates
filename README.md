## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/parthsompura/CVs.git

# Compile any resume
pdflatex Parth_CV_v2.tex
```

## 🎯 Key Features

- **📊 ATS-Optimized** — Clean parsing by Applicant Tracking Systems
- **🎨 Multiple Layouts** — Two-column (Deedy), single-column (classic), compact
- **🎯 Role-Specific Tailoring** — Keyword-optimized for specific job descriptions
- **⚡ One-Page Format** — Concise, high-impact content
