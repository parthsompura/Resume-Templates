## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/CVs.git

# Compile any resume
pdflatex Parth_CV_v2.tex

# Or use your favorite LaTeX editor (Overleaf, TeXShop, VS Code + LaTeX Workshop)
```

## 🎯 Key Features

- **📊 ATS-Optimized** — Clean parsing by Applicant Tracking Systems
- **🎨 Multiple Layouts** — Two-column (Deedy), single-column (classic), compact
- **🎯 Role-Specific Tailoring** — Keyword-optimized for specific job descriptions
- **📝 Cover Letters** — Matching cover letters for each application
- **⚡ One-Page Format** — Concise, high-impact content

---

## 🛠️ How to Tailor for Your Role

1. Pick the closest base version (`v1`, `v2`, or `v3`)
2. Copy and rename: `cp Parth_CV_v2.tex Parth_CV_YourCompany.tex`
3. Update skills & bullet points with keywords from the job description
4. Compile and verify ATS compatibility
